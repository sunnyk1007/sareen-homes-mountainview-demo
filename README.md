# Sareen Homes at Mountain View — GitHub-ready demo site

This is a static demo site built to help you pitch the website job.

## Files

- `index.html` — the main site
- `styles.css` — styling
- `script.js` — interactivity and rendering
- `site-data.js` — the main content file to edit later
- `assets/hero-townhomes.svg` — custom exterior hero illustration
- `assets/overview-townhomes.svg` — custom supporting exterior illustration
- `.nojekyll` — keeps GitHub Pages from doing Jekyll processing

## Fastest way to customize later

Open `site-data.js` and edit:

- brand name
- community name
- location
- hero headline and copy
- price / home count / delivery timing
- feature text
- model names and specs
- location highlights
- contact section copy

You can also replace the image paths in `site-data.js` with your final renderings or photography.

## GitHub Pages deployment

1. Create a GitHub repo.
2. Upload all files from this folder to the repo root.
3. Commit and push.
4. In the repo settings, turn on Pages for the main branch / root.
5. Wait for GitHub to publish the site.

## Notes

- The form is a demo only and does not submit anywhere yet.
- The model names, plans, site plan, and some copy are placeholders.
- The site is designed so you can keep the structure and swap in final content later.
